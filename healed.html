Mental Health Support System

A digital platform that helps students manage anxiety and depression using AI and cognitive assessments. Providing accessible support without stigma.

[ ERR_SYS ] The Problem

  • Students increasingly face anxiety and depression
  • Academic pressure and social stress are constantly rising
  • Professional mental health support is often not easily accessible
  • Many students hesitate to seek help due to stigma

>_ Insight: Anxiety affects a vast majority of students, with depression rates steadily rising across campuses globally.

[ SYS_OK ] The Solution

  • Provides an AI-based mental health support and companion system
  • Uses scientifically backed cognitive tests to analyze mental conditions
  • Offers highly personalized insights and actionable guidance
  • Creates a safe, private, and accessible support system available 24/7

Core Features

  • AI Chatbot: Intelligent mental health conversational agent
  • Cognitive Tests: Built-in assessments (Stroop, N-Back, DTMT)
  • Cognitive Analysis: Deep memory and attention evaluation
  • Personalized Insights: Custom mental health reports
  • Tracking Dashboard: Visual progress tracking over time
  • Pattern Detection: Identifies stress and mood fluctuations
  • Smart Recommendations: Data-driven wellness suggestions
  • Wellness Interface: Calm, user-friendly, and safe UI

Assessment System – Cognitive Analysis

This system evaluates mental health using scientifically designed cognitive tests. Each test measures specific brain functions like memory, attention, and thinking ability.

🧠 Stroop Test

Definition
A psychological test used to measure attention control and cognitive interference.
Purpose
To check how well a user can focus when there is conflicting information.
Simulation Demo
Select the COLOR, not the word.
RED
GREEN

🔄 N-Back Test

Definition
A memory test that evaluates working memory and continuous attention.
Purpose
To measure how well a user can remember, update, and process information.
Simulation Demo
Identify when current letter matches the previous one.
A → B → A → C → A

⚡ DTMT

Definition
Digital Thinking & Memory Test to evaluate thinking ability and memory recall.
Purpose
To analyze how quickly and accurately a user processes sequential information.
Simulation Demo
Identify the pattern and find the next number.
1 → 3 → 5 → ?

Result Analysis

  • System securely collects test results
  • Measures processing speed, accuracy, and consistency
  • Detects early signs of stress or cognitive difficulty
  • Converts complex neuro-data into understandable insights

User Output

  • Clear score breakdown for each cognitive test
  • Visual performance graph over time
  • Identified strength & weakness areas
  • Actionable suggestions for cognitive improvement
